# Pulse100

A bold, focused dashboard for the top 100 crypto assets by market cap.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Environment (Optional)

To reduce rate limits, add a CoinGecko Demo API key:

```
COINGECKO_API_KEY=your_key_here
```

Place it in `.env.local`.
